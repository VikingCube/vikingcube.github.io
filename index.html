FEATURES

  • 8 Channels  
    Supports 10kOhm NTC thermistors
  • USB Serial interface   
    Standard CDC Class,  with virtual serial port. The protocol is line based json.
  • Self calibration
    The internal temperature sensor triggers self calibrations of the AD converters when the temperature of the uC changes.
